# Galaxy Informatics Group (GIG)
<p align="center">
  <img src="gig_logo.png" alt="Galaxy Informatics Group logo" width="250">
</p>

The **Galaxy Informatics Group (GIG)** aims to advance our understanding of galaxies in the era of data-intensive astronomy through reproducible, data-driven research. This is the GIG code hub, where we build open, cutting-edge software for galaxy science powered by machine learning and artificial intelligence (ML/AI).
